# n8n

A self-hostable workflow automation tool. This repository contains the source for n8n adapted for the alesct94-dev organization.

## Features

- Visual workflow editor
- Connects to various services via nodes
- Extensible via custom nodes and integrations

## Getting Started

1. Clone the repository:

   git clone https://github.com/alesct94-dev/n8n.git

2. Install dependencies (example using npm):

   cd n8n
   npm install

3. Start the app in development mode:

   npm run dev

Note: This README provides a minimal starting point. Refer to the project's documentation or internal docs for full setup and production deployment instructions.
